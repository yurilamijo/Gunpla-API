# Gunpla API
Project to improve my Python skills.
API build with Flask-RESTfull

## Getting Started

### MacOS
Setup virual enviroment
```
pip install virtualenv

virtualenv venv

# Mac OS
source ./venv/bin/activate

# Windows
.\venv\Scripts\activate
```

Install packages
```
pip freeze > .\requirements.txt
```

Run it
```
python app.py
```
